🎓 Smart Academy API

AI-powered global education platform — Complete API Documentation

Welcome to Smart Academy API

Smart Academy is an AI-powered global education platform offering courses, events, jobs, blog, community, and AI-powered search. This API provides comprehensive endpoints for managing all platform features.

Base URL: https://api.smart-academy.com/api

Response Format: JSON
Authentication: Check documentation for required headers

📊 Health & Statistics

GET
/healthz
Check API health status
Response: { status: "ok" }
GET
/stats
Get platform statistics
Returns: Total students, courses, instructors, certificates, etc.

👥 Users Management

GET
/users
List all users
Query: role (optional)
POST
/users
Create new user
Required: name, email
Optional: role, avatar, bio, skills, institution, country
GET
/users/:id
Get user profile
Param: id (user ID)
GET
/users/dashboard/:id
Get user dashboard data
Returns: Courses, certificates, streak, recommendations

📚 Courses

GET
/courses
List all courses
Query: category, level, search
POST
/courses
Create new course
Required: title, description, instructor
Optional: category, level, price
GET
/courses/:id
Get course details
Param: id (course ID)

✅ Enrollments

GET
/enrollments
Get user enrollments
Query: userId
POST
/enrollments
Enroll in a course
Required: userId, courseId
PUT
/enrollments/:id
Update enrollment progress
Optional: progress, completedAt

📅 Events

GET
/events
List all events
Query: upcoming (true/false)
POST
/events
Create new event
Required: title, hostName, scheduledAt
Optional: description, meetingUrl, price
GET
/events/:id
Get event details
Param: id (event ID)

💼 Jobs Board

GET

Search jobs
Query: search, type (full-time, part-time, etc.)
POST
/jobs
Post new job
Required: title, company, description
Optional: salary, location, requirements, skills
GET
/jobs/:id

📝 Blog

GET
/blog
Get blog posts
Query: category, limit
POST
/blog
Create blog post
Required: title, content, authorId, authorName
Optional: category, thumbnail, tags, readMinutes
GET
/blog/:id

💬 Community

GET
/community
Get community posts
Ordered by latest first
POST
/community
Create community post
Required: content, authorId, authorName
Optional: topic, authorRole, authorAvatar
POST
/community/:id/like
